# ADK

## 定义
ADK（Agent Development Kit）是 [[entities/eino|eino]] 框架中用于 Agent 开发的核心模块。它提供了一套完整的工具和接口，帮助开发者基于 eino 框架构建 AI Agent 应用。ADK 与 [[concepts/Compose|Compose]] 模块协同工作，前者专注于 Agent 开发能力，后者负责组件编排，两者共同构成 eino 框架的核心能力。

## 关键特征
- 作为 eino 框架的组成部分，专注于 Agent 开发能力
- 提供完整的工具和接口支持 Agent 构建
- 与 Compose 模块协同工作，实现开发与编排的分工
- 支持构建 [[concepts/ReAct-Agent|ReAct Agent]] 等常见应用模式

## 应用
- 基于 eino 框架构建 AI Agent 应用
- 开发 ReAct Agent 等常见 Agent 模式
- 为开发者提供 Agent 开发所需的工具集和接口
