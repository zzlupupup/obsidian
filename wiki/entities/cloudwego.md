# CloudWeGo

## 描述
CloudWeGo 是一个专注于云原生中间件领域的开源组织，致力于开发和维护企业级云原生中间件框架。该组织是 [[entities/eino|eino]] 框架的开发者，eino 是其旗下的企业级云原生中间件框架。CloudWeGo 的 GitHub 仓库托管了 eino 项目，仓库地址为 https://github.com/cloudwego/eino。除了核心框架 eino，CloudWeGo 同时维护了 [[entities/eino-ext|eino-ext]] 扩展库和 [[entities/eino-examples|eino-examples]] 示例仓库，与 eino 核心框架配合使用，形成完整的开发生态系统。
