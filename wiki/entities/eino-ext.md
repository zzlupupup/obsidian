# eino-ext

## 描述
eino-ext 是 [[entities/eino|eino]] 框架的扩展组件项目，其 GitHub 仓库托管于 https://github.com/cloudwego/eino-ext。该项目旨在为 eino 框架提供额外的扩展组件和功能模块，以补充和增强核心框架的能力。在 [[entities/eino-examples|eino-examples]] 的文档中，eino-ext 被列为相关资源，与 eino 框架本体和官方文档共同构成完整的 eino 开发生态系统。该项目由 [[entities/cloudwego|CloudWeGo]] 开源组织维护，是 eino 技术栈的重要组成部分。
