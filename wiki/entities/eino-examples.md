# eino-examples

## 描述
eino-examples 是 [[entities/CloudWeGo|CloudWeGo]] 旗下 [[entities/eino|eino]] 框架的官方示例代码仓库，托管在 GitHub 上。该项目为开发者提供了丰富的示例代码，涵盖 ADK、Compose、Flow、Components、QuickStart 和 DevOps 六大类别。每个示例都针对特定场景设计，如 Agent 创建、编排、多 Agent 协作、人机协作等，帮助开发者快速理解和使用 eino 框架。项目目录结构清晰，按功能模块组织，是学习 eino 框架及其扩展库 [[entities/eino-ext|eino-ext]] 的重要参考资源。通过这些示例，开发者可以快速上手 [[concepts/Compose|Compose]] 编排能力以及 [[concepts/ReAct-Agent|ReAct Agent]] 等核心模式。
