# Manus Agent

## 描述
Manus Agent 是基于 [[entities/eino|eino]] 框架实现的 Agent 示例项目，参考了 [[entities/OpenManus|OpenManus]] 项目的实现思路。它是 [[entities/eino-examples|eino-examples]] 仓库中提供的完整应用示例之一，旨在展示如何使用 eino 框架构建 Agent 应用。该示例位于 eino-examples 仓库的 `flow/agent/manus` 路径下，作为 eino 框架的实践案例，帮助开发者理解如何利用 eino 的 Agent 开发能力构建实际的 AI Agent 应用。Manus Agent 与 [[entities/Deer-Go|Deer-Go]] 同属 eino-examples 中的 Agent 示例项目，共同构成了 eino 框架的示例生态。
