# Deer-Go

## 描述
Deer-Go 是基于 [[entities/eino|eino]] 框架构建的完整应用示例项目，与 [[entities/manus-agent|manus-agent]] 一起作为 eino 框架的实践案例被提及。这些示例代码可在 [[entities/eino-examples|eino-examples]] 仓库中找到，展示了 eino 框架在实际应用中的使用方式。作为 eino 框架的示例项目，Deer-Go 帮助开发者理解框架的组件编排和 Agent 开发能力。该项目体现了 [[concepts/adk|ADK]] 相关的 Agent 开发理念，为开发者提供了可参考的完整实现。
