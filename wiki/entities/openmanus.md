# OpenManus

## 描述
OpenManus 是一个开源项目，为 [[entities/manus-agent|manus-agent]] 示例提供了设计理念和功能参照。在 [[entities/eino|eino]] 框架中，Manus Agent 示例基于 OpenManus 项目的理念和设计进行实现，展示了如何使用 eino 框架的 Agent 开发能力来复现 OpenManus 的功能。作为参考项目，OpenManus 为 eino 框架的 Agent 开发实践提供了设计灵感和功能参照，帮助开发者理解如何基于现有开源项目的思路构建自己的 Agent 应用。
